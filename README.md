# ATS-Score-Checker

An AI-powered ATS Resume Score Checker built with Next.js.

## Features

- ATS compatibility analysis
- Resume scoring
- AI-powered improvement suggestions (Gemini)
- PDF and DOCX resume parsing
- Modern responsive UI

## Tech Stack

- Next.js
- React
- JavaScript
- Tailwind CSS
- Gemini API
- pdf-parse
- mammoth

## Run Locally

```bash
npm install
npm run dev
```

Open:

http://localhost:3000
